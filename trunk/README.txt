1. REQUIREMENTS
maven2 installed on your machine;
java6 installed on your machine

2. BUILD
    mvn clean install

3. RUN
3.1. Server
    java -jar slf4fx-server/dist/slf4fx-server.jar

3.2. Client
Sample client SWF is slf4fx-client/slf4fx-sample/target/MyApplication.swf
Sample SWF could be opened in any browser with Adobe Flash Player plugin installed.
Source code of sample application could be found in slf4fx-client/slf4fx-sample/src/main/flex

Enjoy :)
